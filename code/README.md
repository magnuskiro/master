## Requirements
* Python 2.7
* python-pip

* twython
* Flask
* Flask-SqlAlchemy
* Flask-Assets
* Flask-peewee
* Flask-Script

Pip usage:
```python install <package name>```


## Running the application:
* run the following command in a terminal:  
```python manage.py runserver```

* Open 'localhost:5000/' in a web browser.
